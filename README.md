## Description
Sweet House is a simple static website built using HTML.  
It represents a dessert shop menu that includes ice cream, coffee, bakery items, candies, cakes, and fruit juices.

The project focuses on practicing basic HTML structure and layout using tables and simple styling.

## Technologies Used
- HTML

## Website Sections
- Header with shop name and slogan
- Ice Cream Station
- Fruit Juice Station
- Coffee Station
- Bakery Corner
- Candy Corner
- Cake Display
- Price Table
- Collapsible Full Menu
- Footer

## Features
- Colorful and friendly design
- Multiple food sections
- Menu table with prices
- Collapsible menu using `<details>` and `<summary>`

## Purpose
This project was created for learning and practicing:
- HTML structure
- Tables and layout
- Basic page organization
- Creative UI design using HTML only
